# HDC-APM-5400
APM-5400 Polisher
Download ZIP file of the appropriate software version and extract it to the root directory of an empty USB stick. 
Use "APM-HDC-5400 Polisher Update Master Project-USB.pdf" to update Polishing Machine.

Machine Serial Numbers using this version of software: 3000TS5 and up to 3199TS5.
